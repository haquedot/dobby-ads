# Dobby Ads Full Stack Developer Assignment

Production-ready Google Drive style experience with secure authentication, nested folders, image uploads, and recursive folder size calculation.

## Features
- JWT auth with bcrypt hashing and protected routes
- Nested folder system with dynamic total size tracking
- Image upload, preview, download, and deletion
- Strict access control per user

## Tech Stack
- Backend: Node.js, Express, MongoDB, Mongoose, JWT, 
- Frontend: React, Vite, TailwindCSS, React Router, Axios

## Setup Instructions
### Backend
1. Install dependencies:
   ```bash
   cd backend
   npm install
   ```
2. Configure environment:
   ```bash
   cp .env.example .env
   ```
3. Start server:
   ```bash
   npm run dev
   ```

### Frontend
1. Install dependencies:
   ```bash
   cd frontend
   npm install
   ```
2. Configure environment:
   ```bash
   cp .env.example .env
   ```
3. Start dev server:
   ```bash
   npm run dev
   ```
